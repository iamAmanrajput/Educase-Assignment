# PopX React Project

This repository contains a React-based application built as part of an assignment to create a user interface-driven project using React.

## Features

- User authentication flow with **Sign In** and **Sign Up** forms.
- Form validations and error handling for better user experience.
- Profile management using an **Account Settings** component.
- Responsive and animated UI using **Framer Motion**.
- User data persistence with **localStorage**.
- Client-side routing and navigation.

## Getting Started

1. Clone the repository:
```

git clone <your-repo-url>

```
2. Install dependencies:
```

npm install

```
3. Run the development server:
```

npm start

```
4. Open `http://localhost:5173` to view the app in the browser.

## Structure

- `src/pages` - Contains UI components like SignIn, SignUp, AccountSettings.
- `src/App.jsx` - Main app component with routing.
- `src/main.jsx` - Application entry point, setting up Router and Toaster notifications.
- `public/index.html` - HTML entry point including favicon and metadata.
- `src/index.css` - Base styles for the app.

## Notes

- This project uses React functional components and hooks.
- Animations are implemented with Framer Motion for smooth interactions.
- Form data is validated and stored locally to simulate backend behavior.
- Navigation simulates moving between authentication and account pages.
